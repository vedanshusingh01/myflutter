# Flutter TODO SQLite App

This repository contains a simple Flutter TODO app that demonstrates CRUD operations using sqflite (local SQLite database).

Usage:
1. Replace the repository content or checkout the branch `flutter-todo-crud`.
2. Run `flutter pub get`.
3. Run the app with `flutter run`.

Features:
- Create, read, update, delete todos.
- Mark todos as done.
- Optional due date.

Dependencies:
- sqflite
- path_provider
- path
- intl
